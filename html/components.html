Componentes

El objeto del documento representa su página web. Si desea acceder a cualquier elemento en una página HTML, siempre comienza accediendo al objeto del documento. A continuación se muestran algunos ejemplos de cómo puede utilizar el objeto de documento para acceder y manipular HTML.


Encontrar elementos HTML


Método Descripción
document.getElementById(id) Encuentra un elemento por el ID
document.getElementsByTagName(name) Encuentra los elementos por el nombre de la etiqueta
document.getElementsByClassName(name) Encuentra por el nombre de la clase


Cambio de elementos HTML


Propiedad Descripción
element.innerHTML = new html content Cambia el innerHTML de un elemento
element.attribute = new value Cambia el valor del atributo de un elemento HTML
element.style.property = new style Cambia el estilo de un elemento HTML
Método Descripción
element.setAttribute(attribute, value)     Cambia el valor del atributo de un elemento HTML


Adición y eliminación de elementos


Método Descripción
document.createElement(element) Crea un elemento HTML
document.removeChild(element) Elimina un elemento HTML
document.appendChild(element) Añade un elemento HTML
document.replaceChild(new, old) Reemplaza un elemento HTML
document.write(text) Escribir en el flujo de salida HTML


Adición de controladores de eventos


Método Descripción
document.getElementById(id).onclick = function(){code} Agregar código de controlador de eventos a un evento onclick


Encontrar objetos HTML


Propiedad Descripción
document.baseURI Devuelve el URI base absoluto del documento.
document.doctype Devuelve el tipo de documento del documento.
document.documentElement Devuelve el elemento <html>
document.documentMode Devuelve la URI del documento
document.embeds Devuelve todos los elementos <embed>
document.head Devuelve todos los elementos <head>
document.implementation Devuelve la implementación DOM
document.inputEncoding Devuelve la codificación del documento (conjunto de caracteres)
document.lastModified Devuelve la fecha y hora en que se actualizó el documento.
document.readyState Devuelve el estado (de carga) del documento
document.scripts Devuelve todos los elementos <script>
document.strictErrorChecking Devuelve si se aplica la comprobación de errores
document.title Devuelve todos los elementos <title>
document.URL Devuelve la URL completa del documento
