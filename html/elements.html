Elementos

A menudo, con JavaScript, se desea manipular elementos HTML. Para hacerlo, primero debemos que encontrar los elementos. Hay varias maneras de hacer esto:

  • Encontrar elementos HTML por ID.
  • Encontrar elementos HTML por nombre de etiqueta.
  • Encontrar elementos HTML por nombre de clase.
  • Encontrar elementos HTML mediante selectores CSS.
  • Encontrar elementos HTML por colecciones de objetos HTML.

Nota: El método document.getElementById() solo devuelve el primer elemento con el ID especificado. Si hay más de un elemento con el mismo ID, este método devolverá el primer elemento en el documento.
