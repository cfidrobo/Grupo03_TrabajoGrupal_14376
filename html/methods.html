Métodos

Se puede acceder al HTML DOM con JavaScript y con otros lenguajes de programación.
En el DOM, todos los elementos HTML se definen como objetos .
La interfaz de programación son las propiedades y métodos de cada objeto.
Una propiedad es un valor que puede obtener o establecer como cambiar el contenido de un elemento HTML.
Un método es una acción que puede realizar puede ser agregar o eliminar un elemento HTML.

Ejemplo:
Para ejemplo vamos a usar el siguiente código HTML: Consiste en usar el metodo getElementById junto con el metodo innerHTML para cambiar el contenido de un elemento HTML.

  • La propiedad innerHTML es útil para obtener o reemplazar el contenido de los elementos HTML.

Tipos de Métodos

  • document.getElementById(id)

    • La propiedad id de la interfaz Element representa el identificador del elemento, reflejando el atributo global id. Si el valor de id no es la cadena vacía, debe ser único en un documento.
    La identificación se usa a menudo con getElementById() para recuperar un elemento en particular. Otro caso común es usar la ID de un elemento como selector al diseñar el documento con CSS.

  • document.getElementsByClassName(nombreDeClase)

    • El método getElementsByClassName de la interfaz Document devuelve un objeto similar a una matriz de todos los elementos secundarios que tienen todos los nombres de clase dados.
    Cuando se llama al objeto del documento, se busca en el documento completo, incluido el nodo raíz. También puede llamar a getElementsByClassName() en cualquier elemento; devolverá solo los elementos que son descendientes del elemento raíz especificado con los nombres de clase dados.

  • document.getElementsByTagName(nombreDeEtiqueta)

    • El método getElementsByTagName de la interfaz Document devuelve una HTMLCollection de elementos con el nombre de etiqueta dado.
    Se busca en el documento completo, incluido el nodo raíz. La HTMLCollection devuelta está activa, lo que significa que se actualiza automáticamente para mantenerse sincronizada con el árbol DOM sin tener que volver a llamar a document.getElementsByTagName().

  • document.querySelector(selectorCss)

    • El método de documento querySelector() devuelve el primer elemento dentro del documento que coincide con el selector o grupo de selectores especificado. Si no se encuentran coincidencias, se devuelve nulo.

  • document.querySelectorAll(selectorCss)

    • El método Document querySelectorAll() devuelve una NodeList estática (no activa) que representa una lista de los elementos del documento que coinciden con el grupo de selectores especificado.
